# FUTURE_FS_01 — Portfolio Website

A responsive single-page portfolio website built with plain **HTML**, **CSS**, and **JavaScript**.

## Live Demo

- https://gangastack.netlify.app/

## Preview / Run

- Option 1 (simplest): open `index.html` in your browser.
- Option 2 (recommended): use a local dev server (VS Code Live Server, or any static server) for best results.

## Project structure

- `index.html` — page markup and sections
- `style.css` — styling and animations
- `script.js` — interactions (custom cursor, scroll animations, hamburger menu, smooth scroll, toast)
- `portrait.png` — image asset (if used)

## Features

- Sticky navigation with mobile hamburger menu
- Smooth anchor scrolling
- IntersectionObserver “fade-up” reveal animation
- Custom cursor (desktop only)
- Optional contact form toast

## Notes

This repo is meant to be a static website (no build step, no framework).
